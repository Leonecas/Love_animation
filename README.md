# Love Heart
2/2026

A simple HTML + CSS + JavaScript animation that displays floating **“I love you”** text moving in a heart shape.

## Demo
Open `love.html` in your browser.

## How to Run
1. Download or clone this repo  
2. Open `love.html` in any modern browser (Chrome, Firefox, Edge, etc.)

## Customize
You can easily edit:
- the text (`"I love you"`)
- the color (`#ea80b0`)
- the number of words (`total`)
- the size of the heart (`scale`)
- the speed (`word.t += 0.004`)
